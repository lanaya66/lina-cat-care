# 🚀 快速启动指南

## 当前状态

✅ 项目已创建完成  
✅ 依赖已安装  
✅ 开发服务器已启动在 http://localhost:3000

## 下一步操作

### 1️⃣ 配置 Supabase（必须）

**为什么需要？** 用于云端数据存储和多设备同步

**步骤：**
1. 打开 `SETUP.md` 文件
2. 按照详细步骤创建 Supabase 账号和项目
3. 创建 `.env` 文件并填入配置
4. 在 Supabase 中执行 SQL 创建数据库表

**预计时间：** 10-15 分钟

---

### 2️⃣ 测试应用

配置完成后：

1. **重启开发服务器**（如果已关闭）：
   ```bash
   npm run dev
   ```

2. **打开浏览器**：
   - 访问 http://localhost:3000
   - 注册一个测试账号
   - 尝试添加备餐、记录等功能

3. **手机测试**：
   - 确保手机和电脑在同一 WiFi
   - 手机浏览器访问 http://172.20.10.6:3000
   - 测试移动端体验

---

### 3️⃣ 部署上线（可选）

**推荐方式：Vercel（免费）**

1. 将代码推送到 GitHub
2. 访问 https://vercel.com 并登录
3. 导入 GitHub 仓库
4. 添加环境变量（Supabase 配置）
5. 点击部署

**预计时间：** 5 分钟

部署后会获得一个链接，例如：
```
https://lina-care-record.vercel.app
```

---

### 4️⃣ 添加到手机主屏幕

## 常见问题

### Q: 打开网站显示空白？
A: 检查浏览器控制台是否有错误，通常是 Supabase 配置问题

### Q: 注册后无法登录？
A: 检查 Supabase 是否启用了邮箱确认，建议关闭

### Q: 手机无法访问开发服务器？
A: 确保：
   - 手机和电脑在同一 WiFi
   - 防火墙允许端口 3000
   - 使用正确的 IP 地址
